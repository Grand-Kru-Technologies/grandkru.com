# Copy of Grand Kru landing page

This folder contains a static snapshot of the Canva-hosted "copy-of-grand-kru" site, including its HTML, CSS, and JavaScript. Assets reference the repo's shared `/images` directory, so no build step is required.

## Preview locally

- Open `index.html` directly in a browser, or
- Serve the repo with Vite and visit `http://localhost:5173/copy-of-grand-kru/`.

## Notes

- The contact form is static; submissions are intentionally disabled.
- Mobile navigation closes automatically after selecting a link and keeps the hamburger `aria-expanded` state in sync.
